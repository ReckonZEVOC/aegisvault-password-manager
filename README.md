# 🔐 AegisVault

A full-stack secure password manager built with Spring Boot and React.

## 🚀 Features
- JWT Authentication
- AES Encryption with key derivation
- Secure password vault (Add, Update, Delete)
- Password decryption using master password
- Modern UI with animations

## 🛠 Tech Stack
- Backend: Spring Boot
- Frontend: React
- Database: H2 / MySQL
- Security: AES Encryption

## 📁 Project Structure
secure-password-manager → backend  
password-manager-ui → frontend  

## ▶️ Run Locally

### Backend
cd secure-password-manager  
mvn spring-boot:run  

### Frontend
cd password-manager-ui  
npm install  
npm start
